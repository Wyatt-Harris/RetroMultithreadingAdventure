-Multithreading Adventure   
Summary  
In this action-packed, text-based adventure game, 3 of your favorite characters of all time embark on their own individual journeys, running into friends and enemies along their paths. This game utilizes many advanced object-oriented programming concepts, including multithreading, inheritance, and abstraction. See where your characters' journeys go!

-Features & Use Cases     
The main feature of this game is the use of multithreading to tell multiple action-packed stories at once. The use cases would be anytime a user wants to have some fun!
